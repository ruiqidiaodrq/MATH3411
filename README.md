# MATH3411
### You can try it the following website
https://math-3411.vercel.app/

### Current functions:
- Arithmetic Coding
- Euler's Totient
- ISBN Checker
- Kraft-McMillan
- -LZ78 Encoding
- Prime Checker
- Pseudo-Prime Checker
- Shannon Average Length
- Shannon Entropy

### There will be more functions coming up in the future! Please stay tuned!
